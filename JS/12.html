# 11:ES6新增常用方法

  • 解构赋值
    可以从数组中提取值，按照对应位置，对变量赋值
  1. 数组解构赋值
const arr = [1,2,3]
const [a,b,c] = arr
console.log(a);/* 1 */
console.log(b);/* 2 */
console.log(c);/* 3 */

如果解构不成功，变量的值就等于undefined。

const arr = [1,2]
const [a,b,c] = arr
console.log(a);/* 1 */
console.log(b);/* 2 */
console.log(c);/* undefined */
  1. 对象解构赋值 对象也存在解构赋值
let obj = {
    name: 'Andy',
    age: 34
}
let { name, age } = obj
console.log(name);/* Andy */
console.log(age);/* 34 */

同样如果解构不成功，变量的值就等于undefined。

let obj = {
    name: 'Andy',
    age: 34
}
let { name, age , male} = obj
console.log(name);/* Andy */
console.log(age);/* 34 */        
console.log(male);/* undefined */
  1. 字符串解构赋值 字符串也可以解构赋值。这是因为此时，字符串被转换成了一个类似数组的对象。
const [a, b, c, d, e] = 'hello';
a // "h"
b // "e"
c // "l"
d // "l"
e // "o"
  • 箭头函数
    ES6中,箭头函数可以代替通函数作为函数声明是代码更加简洁
/* 普通函数 */
const fn = function (a, b) {
    console.log(a, b);
}
/* 箭头函数 */
const fn1 = (a, b) => {
    console.log(a, b);
}
  • 如果函数体只有一句代码且代码执行结果就是返回值那么可以省略return和大括号
const fn1 = (a, b) => a + b
  • 如果传递参数只有一个可以省略小括号
const fn1 = a => a

WARNING

是用箭头函数的注意事项
(1)函数体内的this对象，就是定义时所在的对象，而不是使用时所在的对象。
(2)this对象的指向是可变的，但是在箭头函数中，它是固定的。
(3)不可以当作构造函数，也就是说，不可以使用new命令，否则会抛出一个错误。
(4)不可以使用arguments对象，该对象在函数体内不存在。

  • 剩余参数
    剩余参数允许我们将一个不定数量的参数表示为一个数组
const fn = (a,...b) =>{
   console.log(a);/* 1 */
   console.log(b);/* [2,3] */
}
fn(1,2,3)

剩余参数和解构配合使用

const arr = [1,2,3]
const [a,...b] = arr
console.log(a);/* 1 */
console.log(b);/* [2,3] */
  • 扩展运算符
    扩展运算符可以将数组拆分成以逗号分隔的参数序列
let arr = [1,2,3]
console.log(...arr);/* 1 2 3 */

扩展运算符应用

  1. 合并数组
// 方法一
let arr1 = [1, 2, 3]
let arr2 = [4, 5, 6]
let arr3 = [...arr1, ...arr2]
console.log(arr3);/* [1, 2, 3, 4, 5, 6] */

// 方法二
let arr1 = [1, 2, 3]
let arr2 = [4, 5, 6]
arr1.push(...arr2)
console.log(arr1);/* [1,2,3,4,5,6] */
  1. 将伪数组转换成真正的数组
<div>1</div>
<div>2</div>
<script>
    const div = document.getElementsByTagName('div')
    console.log([...div]);/* [div, div] */
</script>
  1. 克隆数组
//ES5中克隆数组的写法
const a1 = [1, 2];
const a2 = a1.concat();
a2[0] = 2;
a1 // [1, 2]

const a1 = [1, 2];
// 新写法
const a2 = [...a1];
  • Array.from()
    Array.from()可以把伪数组转换成真正的数组,常用的两种情况是把DOM操作获得的NodeLists转化成真正的数组还有把函数接收的arguments转换成真正的数组
  • Array.of()
    Array.of()将一组数值转换成数组
Array.of(3, 11, 8) // [3,11,8]
Array.of(3) // [3]

Array.of()是为了解决Array()只能在参数大于二时才能返回转换的新数组的问题

  • find()
    用于找出第一个符合条件的数组成员,如果没有找到返回undefined
const arr = [{
  id: 1,
  name: 'Andy'
},
{
  id: 2,
  name: 'Red'
}]
let target = arr.find(value => value.id === 2)
console.log(target);/* {id: 2,name: 'Red'} */
  • findIndex()
    用于找出第一个符合数组成员的位置,如果没找到返回-1
const arr = [5, 6, 7, 89,]
let target = arr.findIndex(value => value > 10)
console.log(target);/* 3 */
  • includes()
    表示给定的数组是否包含给定的值,包含返回true否则返回false
const arr = [1, 2, 3, 4]
let target1 = arr.includes(2)
let target2 = arr.includes(5)
console.log(target1);/* true */
console.log(target2);/* false */
  • startsWith()和endsWith()
    startsWith()表示参数字符串是否在原字符串的头部,返回布尔值 endsWith()表示参数字符串是否在原字符串的尾部,返回布尔值
  • repeat()
    repeat方法表示将原字符串重复n次并返回一个新字符串
