# 10:元素水平,垂直居中的解决方案

# 水平居中

1. 行内元素

text-align: center;

2. 块级元素
(1). margin实现:

margin: 0 auto;

(2). margin-left和定位实现
父元素相对定位,子元素绝对定位,子元素margin-left值为父元素width/2减去自身width/2
如下代码:

<style>
    .father {
        position: relative;
        width: 300px;
        height: 300px;
        background-color: aquamarine;
    }
    .son {
        position: absolute;
        width: 100px;
        height: 100px;
        margin-left: 100px;
        background-color: pink;
    }
</style>

<div class="father">
    <div class="son"></div>
</div>

(3). 子元素声明display: inline-block;,父元素text-align: center;
如下代码

<style>
    .father {
        width: 300px;
        height: 300px;
        text-align: center;
        background-color: aquamarine;
    }
    .son {
        display: inline-block;
        width: 100px;
        height: 100px;           
        background-color: pink;
        }
</style>

<div class="father">
        <div class="son"></div>
</div>

(4). 利用定位和2D转换transform: translateX
父元素相对定位,子元素绝对定位,子元素left值为50%,transform: translateX(-50%);
代码如下:

 .father {
    position: relative;
    width: 300px;
    height: 300px;
    background-color: aquamarine;
    }
.son {
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
    width: 100px;
    height: 100px;
    background-color: pink;
    }

    <div class="father">
        <div class="son"></div>
    </div>

TIP

子元素中left的50%是父级元素width的50%,而 transform: translateX(-50%)中的-50%是相对自身宽度的

(5). Flex布局实现水平居中

display: flex;
justify-content: center;

# 垂直居中

1. 行内元素
(1). 设置line-hight = 元素的height

2. 块级元素
(2). margin-left和定位实现(可参考块级元素水平居中)
(3). 利用定位和2D转换transform: translateY(可参考块级元素水平居中)
(4). 使用Flex布局的justify-content属性或align-items属性(基于主轴方向不同进行选择)
